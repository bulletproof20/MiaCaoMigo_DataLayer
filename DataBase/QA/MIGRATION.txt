QA notes (simplified architecture)
==================================

Tiers
-----
  00_Bootstrap  Master contract after init_qa
  01_Integrity  DB rules (CI core)
  04_Stress     optional volume/concurrency
  05_Manual     academic demos (non-CI)

Removed
-------
  02_Functional, 04_Regression, run_regression, QaManifest

Bootstrap
---------
  init_qa     CI (Master only)
  init_demo   local demos (Master + Demo)

Docker CI
---------
  docker compose -f docker-compose.yml -f docker-compose.qa.yml down -v
  docker compose -f docker-compose.yml -f docker-compose.qa.yml up -d --build

Runners
-------
  .\run_ci.ps1              PowerShell CI gate
  ./run_qa.sh               bash CI gate (GitHub Actions)
  .\run_stress_all.ps1      optional
  .\run_manual_module.ps1   defense
